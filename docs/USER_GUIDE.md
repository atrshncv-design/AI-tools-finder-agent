# Руководство пользователя

## Авторизация

- Нажмите **Войти** в шапке.
- Доступны два способа:
  - OAuth через Kimi Platform
  - Регистрация по email и паролю

## Лента новостей

- Главная страница (`/`) показывает общие ИИ-новости.
- Новые непрочитанные статьи выделены, счётчик отображается в шапке.
- При открытии статьи она автоматически помечается прочитанной.

## Раздел «ИИ-инструменты для научной работы»

- Перейдите в `/science`.
- Фильтруйте новости по направлениям: химия, материаловедение, биология, медицина, физика, инженерия.
- Фильтруйте по типу: новые инструменты, обновления, закрытия, достижения.

## Поиск

- Введите запрос в строку поиска.
- Поиск работает по заголовкам, аннотациям, подробному содержанию и переводам.

## Избранное

- На карточке новости или в детальной странице нажмите значок звезды.
- Избранные статьи доступны в `/favorites`.

## Детальная страница статьи

- Нажмите на заголовок новости или **Подробнее**.
- Показывается краткое саммари, подробное описание и ссылка на оригинал.
- Кнопка **Перевести** генерирует полный перевод статьи на русский язык.

## Темы оформления

- В шапке переключайтесь между светлой и тёмной темой.
- Интерфейс адаптирован для мобильных устройств.

## Уведомления

- Счётчик непрочитанных статей отображается рядом с лентой.
- Нажмите **Отметить все прочитанными**, чтобы сбросить счётчик.
